# BroFit Website Structure - Complete Site Map

## Pages (Standalone Routes)
1. **Home** (`/`) - Main landing page
2. **Workouts** (`/workouts`) - Exercise library with 800+ exercises
3. **Diet Planner** (`/fuel`) - AI-powered meal planning tool
4. **Calculators** (`/calculators`) - BMI, TDEE, 1RM calculators
5. **Pricing** (`/pricing`) - Membership plans and pricing
6. **Quotes** (`/quotes`) - Motivational quotes

## Sections on Home Page (Scrollable via #id)
1. **Hero** - Main banner
2. **Operations** (`#operations`) - Gym schedule and timings
3. **Features** - "What Can You Do Here?" overview
4. **Daily Protocol** (`#protocol`) - Workout of the day
5. **Trainers** (`#architects`) - Meet the coaches
6. **Contact** (`#contact`) - Location, phone, email

## Navigation Links (Hamburger Menu & Footer)
- Home (/)
- Workouts (/workouts)
- Diet Planner (/fuel)
- Calculators (/calculators)
- Pricing (/pricing)
- Quotes (/quotes)

## Footer Presence
All individual pages (workouts, fuel, calculators, pricing, quotes) will have the Footer component for consistent navigation.
